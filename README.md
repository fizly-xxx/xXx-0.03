# xXx 0.03

Автоматизований софт з графічним інтерфейсом для моніторингу та швидкої купівлі скінів (за 0.03) на ринку Standoff 2 через неофіційне API.

## 🚀 Основний функціонал

* **Автопокупка:** Швидке перехоплення предметів за заданою ціною.
* **Web-GUI:** Зручний локальний інтерфейс для керування ботом.
* **Telegram-інтеграція:** Сповіщення про успішні транзакції в Telegram.
* **Гнучкі конфіги:** Налаштування цільових предметів через JSON.
* **Інвентар:** Відстеження куплених предметів.

## 📁 Структура проєкту

* `bot_core.py` / `main.pyw` — основна логіка та точка входу.
* `ui/` — фронтенд локального інтерфейсу (HTML, CSS, JS).
* `jsons/` — конфігурації (`config.json`, `skins.json`, `weapons.json`).
* `controllers/` — модулі для взаємодії з API ринку та інвентарю.
* `services/` — бекенд-сервіси (трейдинг, пулінг, Telegram-сповіщення).

## ⚙️ Налаштування та запуск

1. Переконайтеся, що встановлено Python та всі необхідні бібліотеки згідно з вашим середовищем.
2. Запустіть main.pyw
